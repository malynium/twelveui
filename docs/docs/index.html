<script>

</script>

Components

Disclosure

NavMenu
A series of links, which are aligned horizontally by default. When you provide "mobile" classes, the menu collapses and is shown/hidden by icons. The default "closed" icon is a three-line hamburger. The default "opened" icon is an X.

Input

Switch
A pill shaped button, which switches between 'on' and 'off' states.
